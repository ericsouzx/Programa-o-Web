Projetos Recentes

Criação deste site como meu Portfólio

O projeto consiste em um site desenvolvido como parte do processo de aprendizagem em programação web durante a faculdade. O objetivo principal foi aplicar na prática os conceitos estudados em sala de aula, como HTML, CSS e JavaScript. O site foi estruturado com foco em usabilidade e organização de conteúdo, permitindo ao usuário navegar de forma intuitiva pelas páginas. Além da parte técnica, o desenvolvimento envolveu boas práticas de design, uso de responsividade para adaptação a diferentes dispositivos e aplicação dos princípios de acessibilidade. O projeto serviu como base para consolidar conhecimentos e preparar para desafios mais complexos no desenvolvimento de aplicações web.
